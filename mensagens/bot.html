🚀 Instalação Rápida (Um Comando)

cd ~ && wget https://code.codehub.ct.ws/mensagens/bot/bot-serve.zip && unzip -o bot-serve.zip -d bot-terminal && cd bot-terminal && npm install && node bot.js

✅ Cole este comando no terminal do seu servidor Linux e pressione Enter!

💻 Comandos do Sistema

Comando Descrição Ação
node bot.js Iniciar bot em modo interativo
pm2 start bot.js --name bot-terminal Iniciar bot 24/7 com PM2
pm2 stop bot-terminal Parar o bot
pm2 restart bot-terminal Reiniciar o bot
pm2 logs bot-terminal Ver logs em tempo real
pm2 status Verificar status do bot

🔄 Atualizar o Bot

pm2 stop bot-terminal && cd ~ && rm -rf bot-terminal && wget https://code.codehub.ct.ws/mensagens/bot/bot-serve.zip && unzip -o bot-serve.zip -d bot-terminal && cd bot-terminal && npm install && pm2 restart bot-terminal || pm2 start bot.js --name bot-terminal

📝 Editor de Comandos Personalizados

➕ Criar Novo Comando

📚 Exemplos de Comandos

!ping
Teste de conexão com o bot
if (msg.text.toLowerCase() === '!ping') { reply('🏓 Pong! Bot online!'); }
!hora
Mostra a hora atual do servidor
if (msg.text.toLowerCase() === '!hora') { const agora = new Date(); reply('🕐 Hora atual: ' + agora.toLocaleTimeString('pt-BR')); }
!dado
Rola um dado de 6 lados
if (msg.text.toLowerCase() === '!dado') { const resultado = Math.floor(Math.random() * 6) + 1; reply('🎲 Você rolou um ' + resultado + '!'); }
!info
Informações do bot
if (msg.text.toLowerCase() === '!info') { reply('🤖 Bot: ' + bot.name + '\n📅 Online 24/7\n💬 Comandos: !ping, !hora, !dado'); }
!moeda
Cara ou coroa
if (msg.text.toLowerCase() === '!moeda') { const resultado = Math.random() < 0.5 ? 'Cara' : 'Coroa'; reply('🪙 Resultado: ' + resultado + '!'); }

🔧 Solução de Problemas

Erro: "node: command not found"

sudo apt update && sudo apt install nodejs npm -y

Erro: "wget: command not found"

sudo apt install wget -y

Erro: "unzip: command not found"

sudo apt install unzip -y

Erro de permissão

sudo chown -R $USER:$USER ~/bot-terminal
