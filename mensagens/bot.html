📋 Instalação Passo a Passo (Linux)

1

Atualizar Sistema e Instalar Dependências

# Ubuntu/Debian
sudo apt update && sudo apt upgrade -y
sudo apt install -y wget unzip curl

# CentOS/RHEL
sudo yum update -y
sudo yum install -y wget unzip curl

# Fedora
sudo dnf update -y
sudo dnf install -y wget unzip curl

# Arch Linux
sudo pacman -Syu --noconfirm
sudo pacman -S --noconfirm wget unzip curl

# Alpine Linux
apk update && apk upgrade
apk add wget unzip curl
2

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# CentOS/RHEL/Fedora
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -

# Arch Linux
sudo pacman -S --noconfirm nodejs npm

# Alpine Linux
apk add nodejs npm

Verificar instalação:

node --version

4

unzip -o bot-serve.zip -d bot-terminal
cd bot-terminal
5

Instalar Dependências NPM

npm install

node bot.js

⚡ Rodar 24/7 em Servidor Linux

Opção 1: PM2 (Recomendado)

# Instalar PM2
npm install -g pm2

# Iniciar bot
cd ~/bot-terminal
pm2 start bot.js --name bot-server

# Salvar configuração
pm2 save

# Iniciar com sistema
pm2 startup

# Ver status
pm2 status

# Ver logs
pm2 logs bot-server

Opção 2: Systemd (Serviço do Sistema)

# Criar arquivo de serviço
sudo nano /etc/systemd/system/bot-server.service

Cole o conteúdo:

[Unit]
Description=Bot Server 24/7
After=network.target

[Service]
Type=simple
User=$USER
WorkingDirectory=/home/$USER/bot-terminal
ExecStart=/usr/bin/node /home/$USER/bot-terminal/bot.js
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target

Ativar serviço:

sudo systemctl daemon-reload
sudo systemctl start bot-server
sudo systemctl enable bot-server
sudo systemctl status bot-server

Opção 3: Screen

# Instalar screen
sudo apt install screen -y

# Criar sessão
screen -S bot

# Iniciar bot
cd ~/bot-terminal && node bot.js

# Desconectar: Ctrl+A, depois D
# Reconectar: screen -r bot

Opção 4: Forever

# Instalar forever
npm install -g forever

# Iniciar bot
cd ~/bot-terminal
forever start bot.js

# Ver processos
forever list

# Parar bot
forever stop bot.js

📝 Criar Comandos Personalizados

📁 Local do arquivo: ~/bot-terminal/comandos.js

Editar comandos:

nano ~/bot-terminal/comandos.js

Exemplo de comando:

module.exports = [
  {
    name: '!oi',
    code: `
if (msg.text.toLowerCase() === '!oi' || msg.text.toLowerCase() === 'oi') {
  reply('👋 Olá ' + msg.senderName + '! Tudo bem?');
}
    `
  },
  {
    name: '!ping',
    code: `
if (msg.text.toLowerCase() === '!ping') {
  reply('🏓 Pong! Bot online 24/7 no Linux!');

    `
  }
];

⚠️ Após editar, reinicie o bot: pm2 restart bot-server

🔧 Solução de Problemas (Linux)

Erro: "EACCES: permission denied"

sudo chown -R $USER:$USER ~/bot-terminal
chmod -R 755 ~/bot-terminal

Erro: Porta já em uso

# Ver processo na porta
sudo lsof -i :3000
# Matar processo
sudo kill -9 [PID]

Erro: Cannot find module

cd ~/bot-terminal
rm -rf node_modules package-lock.json
npm install

Firewall bloqueando

# Ubuntu/Debian (ufw)
sudo ufw allow 3000

# CentOS/RHEL (firewalld)
sudo firewall-cmd --permanent --add-port=3000/tcp
sudo firewall-cmd --reload

💾 Backup e Restauração

Fazer backup dos bots:

cp ~/bot-terminal/bots_local.json ~/backup-bots-$(date +%Y%m%d).json

Backup dos comandos:

cp ~/bot-terminal/comandos.js ~/backup-comandos-$(date +%Y%m%d).js

Restaurar backup:

cp ~/backup-bots-*.json ~/bot-terminal/bots_local.json
pm2 restart bot-server
