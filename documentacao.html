<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Sistema de Autenticação CodeHUB</title>
  <style>
    body { font-family: Arial, sans-serif; max-width: 800px; margin: 40px auto; padding: 20px; }
    .dados { background: #f4f4f4; padding: 20px; border-radius: 5px; margin-top: 20px; }
    .dados p { margin: 10px 0; font-size: 16px; }
    button { padding: 10px 20px; margin: 5px; cursor: pointer; font-size: 16px; }
    .login-area { background: #e3f2fd; padding: 20px; border-radius: 5px; margin-bottom: 20px; text-align: center; }
    h1 { margin-top: 0; }
  </style>
</head>
<body>
  <div id="loginArea" class="login-area">
    <h2>Faça login para continuar</h2>
    <button onclick="abrirLogin()">Fazer Login</button>
  </div>
  
  <div id="dadosUsuario" style="display: none;">
    <h1>👤 Dados do Usuário</h1>
    <button onclick="sair()">Sair</button>
    <div id="resultado" class="dados">Carregando...</div>
  </div>

  <script>
    const apiUrl = "https://code-hub-eta.vercel.app/api/userkey.js";
    // PEGAR SEU TOKEN EM: https://code.codehub.ct.ws/API/criar.app
    const loginUrl = "https://code.codehub.ct.ws/API/v2/continuar-conta?token=SEU_TOKEN_AQUI";
    
    function abrirLogin() {
      window.open(loginUrl, "_blank");
    }
    
    function buscarDados(userkey) {
      fetch(`${apiUrl}?userkey=${encodeURIComponent(userkey)}`)
        .then(res => res.json())
        .then(data => {
          const nome = data.nome || "Não disponível";
          const email = data.email || "Não disponível";
          
          document.getElementById("resultado").innerHTML = `
            <p><strong>Nome:</strong> ${nome}</p>
            <p><strong>E-mail:</strong> ${email}</p>
            <p><strong>UserKey:</strong> ${userkey}</p>
          `;
          
          document.getElementById("loginArea").style.display = "none";
          document.getElementById("dadosUsuario").style.display = "block";
        })
        .catch(err => {
          document.getElementById("resultado").innerHTML = "Erro: " + err;
        });
    }
    
    function sair() {
      localStorage.removeItem("userkey");
      window.history.pushState({}, "", window.location.pathname);
      document.getElementById("loginArea").style.display = "block";
      document.getElementById("dadosUsuario").style.display = "none";
    }
    
    const urlParams = new URLSearchParams(window.location.search);
    const userkeyFromUrl = urlParams.get('userKey');
    const userkeySalvo = localStorage.getItem("userkey");
    
    if (userkeyFromUrl) {
      localStorage.setItem("userkey", userkeyFromUrl);
      buscarDados(userkeyFromUrl);
      window.history.pushState({}, "", window.location.pathname);
    } 
    else if (userkeySalvo) {
      buscarDados(userkeySalvo);
    }
    else {
      document.getElementById("loginArea").style.display = "block";
      document.getElementById("dadosUsuario").style.display = "none";
    }
  </script>
</body>
</html>

📖 Como Usar o Sistema de Autenticação

🔑 Endpoint da API:
https://code-hub-eta.vercel.app/api/userkey.js

1️⃣ Como Funciona

O sistema de autenticação permite que os usuários façam login usando um UserKey. O fluxo é:

  • Clique em "Fazer Login" para abrir a página de autenticação
  • Após o login, você será redirecionado com o UserKey na URL
  • O sistema captura automaticamente o UserKey e salva no localStorage
  • Os dados do usuário são buscados e exibidos
  • O UserKey é removido da URL por segurança

2️⃣ Buscar Dados do Usuário

GET https://code-hub-eta.vercel.app/api/userkey.js?userkey=SEU_USERKEY

Retorna os dados do usuário incluindo nome, email e token de autenticação.

3️⃣ Obter Token de Login

https://code.codehub.ct.ws/API/criar.app

Acesse este link para criar seu token de autenticação.

💡 Dica: Copie o código HTML acima e substitua "SEU_TOKEN_AQUI" pelo seu token gerado em https://code.codehub.ct.ws/API/criar.app
